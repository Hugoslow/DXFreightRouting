Expected File Format

Your Excel file must have these columns (in any order):

Column Description Example Required
Date Collection date 2026-01-06 or 06/01/2026 Yes
CPID Collection Point ID (must exist in system) CP001 Yes
Parcels Number of parcels 1500 Yes
Trailers Number of trailers 2 Yes
Collection Time Collection time (24hr format) 09:00 or 14:30 Optional

Validation Rules:

  • CPID must exist in the system - unknown CPIDs will be rejected
  • Parcels and Trailers must be positive numbers
  • Duplicate entries (same date + CPID) will be skipped
  • Files can contain data for any date, including future dates
